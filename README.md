# local-music-scene
It Shows resources/media on local music scenes around the world.
The files are from phone recordings that I had collected from some of my musician friends.
Introduction to the Project: 
  I started from 2015 with my interests in local music scenes. 
  I would like to start this project so that I can rationally understand 
  what the world has to offer for people who have interests in music 
  and why it could be interesting.
